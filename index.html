What is vSphere Integrated Containers Engine?

vSphere Integrated Containers Engine is a container runtime for vSphere, allowing developers familiar with Docker to develop in containers and deploy them alongside traditional VM-based workloads on vSphere clusters, and allowing for these workloads to be managed through the vSphere UI in a way familiar to existing vSphere admins.

See vSphere Integrated Containers Engine Architecture for a high level overview.

Get vSphere Integrated Containers Engine

vSphere Integrated Containers Engine source code is available on the VMware vSphere Integrated Containers Engine GitHub source repository.

  • You can download the latest approved build of vSphere Integrated Containers Engine from https://github.com/vmware/vic/releases/.
  • You can download the latest daily (unapproved) build of vSphere Integrated Containers Engine from https://bintray.com/vmware/vic-repo/build/view#.
